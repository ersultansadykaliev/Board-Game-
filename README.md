# 🎮 Telegram Mini App: Настольные Игры (Уголки, Шахматы, Шашки)

Современное веб-приложение (Telegram Mini App) с классическими настольными играми. Проект поддерживает игру как против умного ИИ, так и против реальных людей (PvP) по ссылке-приглашению.

## ✨ Ключевые возможности

- **3 Игры в одном приложении**: Шахматы, Шашки и Уголки.
- **Glassmorphism UI**: Красивый современный интерфейс со стеклянными панелями, градиентами и микро-анимациями.
- **Мультиплеер (PvP)**: Создавайте комнату и отправляйте друзьям ссылку. Состояние синхронизируется через сервер.
- **ИИ-Оппонент (PvE)**: Интеллектуальный бот, использующий алгоритм Minimax с Alpha-Beta отсечением.
- **Сохранение игр (SQLite)**: База данных гарантирует, что ваши партии не пропадут даже при перезагрузке сервера. 

## 🛠 Технологии

- **Backend**: Python 3, Flask, SQLite, Pickle (для сериализации объектов игры).
- **Frontend**: Vanilla JS, CSS3, HTML5, Telegram WebApp API.
- **Архитектура**: Полное разделение игровой логики (`games/`) и веб-сервера (`webapp/`).

## 🚀 Быстрый старт (Локально)

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустите Flask-сервер
```bash
# Перейдите в папку проекта и запустите app.py
python webapp/app.py
```
Сервер запустится на `http://127.0.0.1:5000`.

### 3. Откройте в браузере
Перейдите по адресу `http://127.0.0.1:5000`, чтобы увидеть веб-интерфейс.

## 📱 Интеграция с Telegram

Чтобы приложение работало внутри Telegram как Mini App:
1. Создайте бота через [@BotFather](https://t.me/BotFather).
2. Загрузите код `webapp/` на хостинг (например, PythonAnywhere, Heroku или ваш VPS). Сервер должен поддерживать **HTTPS**.
3. В настройках BotFather привяжите кнопку "Play" к URL вашего хостинга.
4. (Опционально) Запустите `bot.py`, если хотите, чтобы бот сам отправлял приветственные сообщения с кнопкой запуска Mini App.

## 📁 Структура проекта

```
ugolki-bot/
├── games/               # Игровая логика
│   ├── chess/           # Правила, доска и ИИ для Шахмат
│   ├── checkers/        # Правила, доска и ИИ для Шашек
│   └── ugolki/          # Правила, доска и ИИ для Уголков
├── webapp/              # Веб-сервер и Frontend
│   ├── app.py           # Flask-сервер, API эндпоинты, SQLite
│   ├── templates/       # HTML шаблоны
│   └── static/          # CSS (стили) и JS (логика браузера)
├── bot.py               # (Опционально) Файл для Telegram-бота
├── games.db             # (Генерируется) База данных активных игр
├── requirements.txt     # Зависимости Python
└── README.md            # Этот файл
```
