# 🎮 Telegram Бот «Уголки» (Corners / Halma)

Классическая настольная игра «Уголки» в Telegram!  
Переместите все свои фигуры из своего угла в угол противника.

## 🚀 Быстрый старт

### 1. Создайте бота в Telegram
1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный API-токен

### 2. Установите зависимости
```bash
pip install -r requirements.txt
```

### 3. Настройте токен
Вариант A — через переменную окружения:
```bash
# Windows
set UGOLKI_BOT_TOKEN=ваш_токен_от_BotFather

# Linux/Mac
export UGOLKI_BOT_TOKEN=ваш_токен_от_BotFather
```

Вариант B — напрямую в `config.py`:
```python
BOT_TOKEN = "ваш_токен_от_BotFather"
```

### 4. Запустите бота
```bash
python bot.py
```

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и описание бота |
| `/play` | Начать новую игру (выбор режима) |
| `/join` | Присоединиться к PvP игре |
| `/rules` | Правила игры |
| `/cancel` | Отменить текущую игру |

## 🎯 Режимы игры

- **👥 PvP** — два игрока в одном чате
- **🤖 PvE** — игра против искусственного интеллекта

## 🕹️ Как играть

1. Отправьте `/play` и выберите режим
2. Нажмите на свою шашку (⚪ или ⚫) — она подсветится 🟡
3. Нажмите на зелёную клетку 🟢 — сделать ход
4. Побеждает тот, кто первым переместит все фигуры в угол противника!

## 📜 Правила «Уголков»

- Доска 8×8, у каждого игрока 14 шашек в углу
- **Простой ход**: на 1 клетку в любом направлении (8 направлений)
- **Прыжок**: перескок через соседнюю шашку на свободную клетку
- **Цепочка**: несколько прыжков за один ход
- Шашки **не съедаются** — все остаются на доске!
- Лимит: 40 ходов на каждого игрока
